# Pharmacy API
### API for managing your medicines

## Requirements
- nodejs
- npm
- mongodb

## Environment
You should have .env.[environment] file, for example: *.env.test, .env.dev*

Variables that you will need to set you can copy from **.env.example**

## How to run

```bash
npm install

// in development mode
npm run dev

// in production mode
npm start
```

## How to run tests

```
npm test
```
